Cum te conectezi

Patru pași simpli și ești în legendă

01

Java Edition

Ai nevoie de Minecraft Java Edition. Versiunile 1.8 – 1.21 sunt acceptate prin ViaVersion. Serverul rulează pe PaperMC 26.1.2.

02

Multiplayer → Add Server

Din meniul principal apasă Multiplayer → Add Server și completează câmpurile de mai jos.

03

Introdu adresa
